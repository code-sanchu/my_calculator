Functionality

This calculator is non-scientific

takes max 8 digits as number input, including a decimal point

only answer with an absolute value 8 sig figures and less accepted

decimal number answers will be rounded to the 8th digit if greater than 8 digits

negative numbers accepted but no direct way to input them

errors reset the calculator

operators can be changed

operations can be chained

zeros inputted but not preceeded by any other number will function but won't be displayed (sorry have to move on)
